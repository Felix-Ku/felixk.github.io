LeetCode trial - Q12 (Integer to Roman)

2020-06-25

Link to Question

  • Objective: To convert integer into Roman Numeral
  • Level of difficulty: Medium
  • Language used: C++

Result

First trial

  • Runtime: 12 ms (faster than 70.86%) For reference only
  • Memory Usage: 5.9 MB (less than 89.33%) For reference only

Second trial

More intuitive method

  • Runtime: 8 ms (faster than 88.34%) For reference only
  • Memory Usage: 6 MB (less than 78.85%) For reference only

Code

First trial

class Solution {
public:
    string intToRoman(int num) {
        string output="";
        int value[13] = {1000,900,500,400,100,90,50,40,10,9,5,4,1};
        string valueS[13] = {"M","CM","D","CD","C","XC","L","XL","X","IX","V","IV","I"};
        int count=num;
        int pos=0;
        while (count!=0)
        {
            if (count>=value[pos])
            {
                count=count-value[pos];
                output=output+valueS[pos];
            }
            else
                pos++;
        }
        return output;
    }
};

Second trial

class Solution {
public:
    string intToRoman(int num) {
        string output="";

        while (num!=0)
        {
            if (num>=1000)
            {
                num=num-1000;
                output=output+"M";
            }
            else if (num>=900)
            {
                num=num-900;
                output=output+"CM";
            }
            else if (num>=500)
            {
                num=num-500;
                output=output+"D";
            }
            else if (num>=400)
            {
                num=num-400;
                output=output+"CD";
            }
            else if (num>=100)
            {
                num=num-100;
                output=output+"C";
            }
            else if (num>=90)
            {
                num=num-90;
                output=output+"XC";
            }
            else if (num>=50)
            {
                num=num-50;
                output=output+"L";
            }
            else if (num>=40)
            {
                num=num-40;
                output=output+"XL";
            }
            else if (num>=10)
            {
                num=num-10;
                output=output+"X";
            }
            else if (num>=9)
            {
                num=num-9;
                output=output+"IX";
            }
            else if (num>=5)
            {
                num=num-5;
                output=output+"V";
            }
            else if (num>=4)
            {
                num=num-4;
                output=output+"IV";
            }
            else
            {
                num=num-1;
                output=output+"I";
            }
        }
        return output;
    }
};

Notes

  • Kinda like brutal force
  • After observation, Roman numerals changes representation method while:
    • Changes from adding up into substracting down
    • 4 -> IV
    • 9 -> IX
    • 40 -> XL
    • 90 -> XC
    • 400 -> CD
    • 900 -> CM
    • Therefore adding the above into tracking array can enhance direct convertion
